# 模板文件说明

├─main.c
├─pid_app.c
├─pid.c
├─motor.c
├─sensor.c
├─comm.c
└─timer.c

* main.c：主程序，初始化定时器中断和用于与视觉模块通信的串口，两个软件定时器，电机速度初始为0，开启控制部分
* pid_app.c：控制部分的主要函数
* pid.c：pid的底层函数
* motor.c：电机底层函数
* sensor.c：灰度传感器底层函数
* comm.c：通信部分底层函数
* timer.c：软件定时器底层函数


# 电机按键设置初始化：

使用步进电机和驱动：1.8°，16细分，发送3200个脉冲电机转一圈
左电机地址2；右电机地址1
左电机方向CW；右边电机方向：CCW
